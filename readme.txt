== Description ==

A standalone version of Infinite Scroll from Jetpack.

This is an unofficial copy with some modifications:
* Removed support to WordPress built-in themes, such as twentyfifteen.

== Installation ==

Same as Jetpack.

== Frequently Asked Questions ==

= Does this version work with official Jetpack? =

Yes, as long as Infinite Scroll is not enabled in the official one.
